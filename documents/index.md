# Introduction to Rben

### What does it look like?

```js
<Rben
  Globals={['puts "hello"']}
  Cases={['puts "hello"']}
/>
```

### Getting Started

```shell
git clone https://github.com/tseijp/rben
```

Run the development server:

```shell
cd rben
yarn init
yarn start
```

Your site starts at [http://localhost:3000](http://localhost:3000).
